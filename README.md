# vimrc
base https://github.com/amix/vimrc
```
git clone --depth=1 https://github.com/amix/vimrc.git ~/.vim_runtime
sh ~/.vim_runtime/install_awesome_vimrc.sh
```

## Keymap memo
- tab : NERDTreeToggle
- ,nf : NERDTreeFind
- vs : 세로분할
- sp : 가로분할
- Shift+f,b : finder
- F8 : tagbar
- Ctrl+left,right : tab move left, right
- ,f : MRU
